# Computer Science with Python — Class 11 (CBSE)

This repository contains teaching materials and supportive resources for CBSE Class 11 Computer Science (Python), aligned with the 2025-26 syllabus. It is intended as an open educational library for teachers and students to host notes, example programs, and practice exercises.

**Important:** This project does NOT include or redistribute the full textbook PDF or any other copyrighted textbook content. It provides pointers and original teaching material only. For the official textbook, please purchase or obtain a licensed copy from the publisher.

**Textbook reference**: Sumita Arora — "Computer Science with Python" (Class 11). Publisher: Dhanpat Rai & Co. (2025-26 Examination Edition). Sample/archival link: https://pythonclassroomdiary.wordpress.com/wp-content/uploads/2020/11/sumita-arora-class-xi-computer-science-with-python-pdf.pdf

## Core Syllabus Coverage (CBSE 2025-26)
- Unit 1: Computer System Organisation — basics, memory, software types
- Unit 2: Computational Thinking and Programming — Python fundamentals, control flow, data structures (strings, lists, tuples, dicts), debugging, sorting
- Unit 3: Society, Law, and Ethics — digital footprint, cyber ethics, data protection

## What this repo provides
- **Beginner Level**:
  - `1-beginners-questions.ipynb`: 10 basic Python questions (no solutions).
  - `2-beginner-answers.ipynb`: 10 basic Python questions with solutions.
- **Intermediate Level**:
  - `3-intermediate-questions.ipynb`: 10 intermediate Python questions (no solutions).
  - `4-intermediate-answers.ipynb`: 10 intermediate Python questions with solutions.
- **Advanced Level**:
  - `5-advanced-questions.ipynb`: 10 advanced Python questions (no solutions).
  - `6-advanced-answers.ipynb`: 10 advanced Python questions with solutions.
- Lesson notes and concise summaries for each chapter
- Example Python programs and solved exercises
- Practical exercises and sample viva-voce questions
- Contribution-friendly structure for teachers and students

## Usage
1. Browse folders for chapter-wise examples and practice problems.
2. Copy example Jupyter notebooks or .py files for classroom demonstrations.
3. Open Issues or Pull Requests to suggest additions or fixes.
